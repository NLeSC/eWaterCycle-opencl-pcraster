Operation

Werking is zoals van de arealimit functie variant met de opmerking dat de areaLimit argument vervangen zijn door de areaAdded argument. Het algoritme zal op basis van "maximum likelyhood" het aureaal ten grootte areaAdded vergroten.

Notes

A cell with missing value on chances is not considered in the operation; it is assigned a missing value on Result. An id_i value of 0 will lead to unclear results.

Group

This operation belongs to the group of Order
