timeplot and display are deprecated

The 'display' and 'timeplot' applications are deprecated and no longer supported. For visualisation use the 'aguila' application instead.

For backwards compatibility 'display' and 'timeplot' are included in the distribution that by default will call aguila. You need to explictly set an option to continue using the old display or timeplot applications. Additional options to display and timeplot are:
--no-aguila-warning: disables the deprecation warning from now on
--aguila-default: enables the deprecation warning from now on
--old-display and --old-timeplot: do not invoke aguila, but display and timeplot respectively
